Lego Fabrication Shop

Ed Emberly's drawing alphabet brings a similar system to drawing. By mastering the "visual alphabet" as Emberly defines it, children lean the fundamental elements from which you can draw anything! In my experience, students armed with this system (we provided the basic shapes cut from construction paper) were able to quickly bring many ideas to form, rather than getting held up on details.
